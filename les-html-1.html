Занятие 1 - Введение в веб-разработку. HTML-страница

Добро пожаловать на первое занятие по веб-разработке! В этом уроке мы поговорим о том, что такое веб-разработка, какие технологии используются и как начать свой путь в этой области.

Что такое веб-разработка?

Веб-разработка - это процесс создания веб-сайтов и веб-приложений. Она включает в себя разработку клиентской и серверной частей веб-приложений, а также их интеграцию и тестирование.

Какие технологии используются в веб-разработке?

Для создания веб-приложений используется множество технологий. Основные из них:

  • HTML - язык разметки для создания содержимого веб-страниц;
  • CSS - язык описания внешнего вида веб-страниц;
  • JavaScript - язык программирования для создания интерактивных элементов;
  • Базы данных - для хранения и обработки данных веб-приложений;
  • Фреймворки и библиотеки - для упрощения и ускорения процесса разработки.

Что такое веб-сайт

Веб-сайт — это объединённая куча веб-страниц (файлов .html) и доступная пользователям по запросу.

Веб-страница — это текстовый документ, написанный на языке понятном браузерам.

Браузер — это программа для просмотра веб-страниц. Она умеет обрабатывать код страницы и отображать его в удобном для пользователя виде.

Самый популярный и по факту который поддерживает все последние обновления и фишки языков для разработки это Google Chrome + он имеет шикарные инструменты для разработчика. Поэтому будем использовать его.

Где хранить HTML-страницу, чтобы она была доступна по всему миру?

Сервер — компьютер, подключённый к Интернету, на котором хранятся файлы сайта. Сервер отправляет эти файлы пользователю по запросу. Пользователь просматривает их с помощью браузера.

Клиент — компьютер, подключённый к Интернету, который обращается за данными к серверу.

Здесь мы будем писать HTML разметку, это тот самый каркас на котором держится структура нашего сайта.

Документ HTML состоит из тегов. Можно вернуться снова в браузер и посмотреть как эти теги выглядят.

HTML (HyperText Markup Language) — набор правил, по которым браузер отличает заголовки от списков, таблицы от картинок и так далее.

Слово «язык» в HTML стоит воспринимать как правила. Сам по себе HTML только размечает данные, но никак с ними не взаимодействует и визуально с ними ничего не делает. Всю работу по выводу текстовых данных и разметки берет на себя браузер.

HTML: Базовая структура

Любой HTML-документ можно разбить на несколько основных составляющих:

  • Внутри тегов <head></head> находится техническая служебная информация которая не видна при отображении сайта в браузере. Здесь мы можем подключить различные стили, скрипты, мета теги и т.п. служебную информацию. Также здесь можно указать заголовок который будет отображаться на вкладке в браузере.
  • Внутри тегов <body></body> мы пишем всё содержимое нашего сайта, которые и будет видно в браузере. 
<!DOCTYPE html>
<html lang="ru">

  <head>
    <meta charset="UTF-8">
    <title>Мой сайт</title>
  </head>

  <body>
    <h1>Иванов Иван Иванович</h1>
    <p>Здесь будет информация обо мне</p>
  </body>

</html>

<!DOCTYPE html> - служебная информация для браузера, где описывается стандарт HTML, который нужно обработать. Текущий стандарт HTML5 поддерживается всеми браузерами, поэтому достаточно указать в первой строке документа <!DOCTYPE html>.

Далее открывается парный тег <html></html> с атрибутом lang="ru". Указание языка также необходимо для браузеров, особенно если сайт рассчитан на массовую аудиторию, в том числе и за рубежом.

HTML: Мета теги

Мета теги — служебные теги в разметке HTML. Они предназначены для указания сведений поисковым роботам и браузерам.

Тег title

За заголовок во вкладке браузера отвечает парный тег <title>, внутри которого содержится название страницы.

Обычно советуют использовать заголовок от 50 до 80 символов. Такое ограничение существует из-за того, что поисковые системы не могут показать больше символов в качестве заголовка страницы в результатах поисковой выдачи.

Заголовок внутри тега <title> отобразится во вкладке в браузере. 

<title>Мой сайт</title>

Тег meta

Большинство мета тегов записываются с использованием непарного тега <meta>. Он принимает несколько основных атрибутов, которые помогают установить метаинформацию

Описание

Описание часто используется при добавлении ссылки на страницу в социальных сетях. Эта информация используется поисковыми системами при выводе страницы в результатах поиска.

Чтобы установить описание, используется тег <meta> с двумя атрибутами:

  • name="description". Атрибут name указывает на то, какой тип метаинформации описывается в теге
  • content. Атрибут content указывает саму информацию 
<meta name="description" content="Это личный сайт портфолио от web-разработчика">

Ключевые слова

В разработке для страниц часто указывают ключевые слова. Они помогают роботам корректнее собирать информацию о странице, что хорошо сказывается на продвижении сайта в интернете.

В качестве значения атрибута name используется keywords. Сами ключевые слова перечисляются через запятую 

<meta name="keywords" content="web-разработка, верстка, HTML, CSS">

Кодировка страницы

Существует большое количество кодировок — таблиц символов, по которым происходит поиск буквы или символа. Раньше проблема кодировок стояла очень остро — почти для каждого языка использовалась уникальная кодировка, из-за чего открыть документ на другом языке было невозможно.

Считается хорошим тоном, в качестве мета тега, указывать кодировку страницы.

Для этого указывается только один атрибут — charset, значением которого выступает название кодировки. В 99% случаев это UTF-8. 

<!-- Кодировка страницы UTF-8 -->
<meta charset="UTF-8">

Комментарии в коде

Что такое <!-- --> ? 

<!-- Кодировка страницы UTF-8 -->

Так в HTML создаются комментарии.

Привыкаем сразу пользоваться комментариями, мы это будем делать часто 😎

HTML разметка и вид сайта в браузере на текущий момент 

<!DOCTYPE html>
<html lang="ru">
  
  <head>
    <!-- Кодировка страницы UTF-8 -->
    <meta charset="UTF-8">
    <!-- Описание страницы для поисковой оптимизации -->
    <meta name="description" content="Это сайт портфолио от web-разработчика">
    <!-- Ключевые слова страницы для поисковой оптимизации -->
    <meta name="keywords" content="web-разработка, верстка, HTML, CSS">

    <!--Заголовок на вкладке в браузере -->
    <title>Мой сайт</title>
  </head>

  <body>
    <h1>Сергиенко Роман Андреевич</h1>
    <p>Здесь будет информация обо мне</p>
  </body>

</html>
